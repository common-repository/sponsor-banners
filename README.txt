Manage banners for personal blog and web sites.

== Description ==

Simple but useful and effective manager of banners that will allow you to easily handle many banners, categorize and 
display them in the front end.

Feature
*   save banners images and links
*   save dimensions of banners
*   multicategories of banners
*   widget to display banner in front end


== Installation ==


1. Upload `sponsor.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use widget to display banners in front-end
